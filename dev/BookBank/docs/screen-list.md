# 画面一覧（Screen List）

| 画面ID | 画面名 | 画面の役割 | 主な要素 |
|---|---|---|---|
| S-01 | ホーム画面 | アプリ起動後の最初の画面 | ・累計情報の表示（金額・冊数）<br>・書籍登録への導線<br>・通帳画面への導線 |
| S-02 | 書籍登録画面 | 書籍を検索し、通帳へ登録するための中心画面 | ・書籍検索キーワード入力<br>・検索実行操作<br>・未登録書籍のみ表示する条件指定<br>・検索結果一覧表示<br>・書籍選択操作<br>・検索結果が存在しない場合の手動登録導線 |
| S-03 | 手動登録画面 | 検索で登録対象が見つからない場合に、書籍情報を直接入力する代替登録画面 | ・書籍情報入力（タイトル・著者・金額） |
| S-04 | 通帳画面 | 登録済み書籍を金額・冊数の視点で一覧確認する画面 | ・登録済み書籍の一覧表示（リスト形式）<br>・書籍の基本情報表示（サムネイル・タイトル・著者・金額・日付） |
| S-05 | 本棚画面 | 登録済み書籍を視覚的に整理・閲覧するための画面 | ・登録済み書籍の一覧表示（アイコン形式）<br>・書籍サムネイル表示 |
| S-06 | 書籍詳細画面 | 書籍単位での情報確認・個人メモ管理を行う画面 | ・書籍詳細情報の表示<br>・メモの入力／編集<br>・お気に入り状態の切り替え<br>・所属口座の表示 |
| S-07 | 口座管理画面 | 通帳（口座）の作成および使用中口座の切り替えを行う管理画面 | ・口座の新規作成<br>・使用中口座の切り替え |
| S-08 | ポートフォリオ画面 | 読書によって蓄積された知的資産を、期間・構成の視点で可視化する分析画面 | ・累計金額／冊数の推移表示<br>・年別／月別の集計表示<br>・口座別の構成比表示 |

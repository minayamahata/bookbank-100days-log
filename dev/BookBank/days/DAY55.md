## 動画の内容
100 日チャレンジDAY55、ようやく今日から実装に入ります。まずは事前設計フェーズで作ってきた資料をマークダウン形式に書き換えて DOCS フォルダに格納しました。私の場合は Figma や FigJam を使って作ったので、わざわざマークダウンに書き換える作業が発生しましたが、Figma のDevモードとプラグインを組み合わせればできるかもなぁなんてほんのり思いました。
DOCS の事前設計書を参照して、 iOS の Swiftdataを使いデータモデルを作ります。ここで迷ったのが、カーソル上のターミナルでclaudeコマンドを呼び出すのか、それともClaudeエージェントに話しかけるのか。
この違いがまだよく分かっていないので、 GPT に言われた通りターミナルで行いました。ターミナルの文面は一ミリも分からないので、スクショを撮って都度確認しながら進めます。私は五つのモデルで設計していたのですが、Claudeが作ったのは三つでした。
補完する形で仕上げてくれたようですが。「だったら事前設計やる意味なかったじゃん」と伝えたところ、「あなたが事前設計をしたからこそ最高のアウトプットが出せたんですよ」とマインドコントロールされそうになりました。
